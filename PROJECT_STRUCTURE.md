# Project Structure

## Directories
- `/html` — HTML templates and pages
- `/css` — Stylesheets
- `/js` — JavaScript files

## Entry Points
- `/html/index.html` — main page
- `/css/styles.css` — shared styles
- `/js/app.js` — logic, parsing, deduplication, UI rendering, search

## Notes
- All UI components share the same base page; no separate component files
- The UI Components Guide has been removed entirely per Search team requirements
- Sitemap updated at `/sitemap.xml` pointing to `/html/index.html`
